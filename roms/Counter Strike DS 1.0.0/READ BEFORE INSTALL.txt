- For EACH new update, replace all files with new files.


- If the game can't load :
On Nintendo DS classic :
Try to put the soundbank.bin file in the root (root example : G:\soundbank.bin) of you sd card, or try another linker (R4, M3...)

On Nintendo 3DS/2DS
Use TWiLight Menu++ : https://wiki.ds-homebrew.com/twilightmenu/installing-3ds
Put all the files in "roms\nds"

- I have no sound
On Nintendo DS classic :
Try to put the soundbank.bin file in the root (root example : G:\soundbank.bin) of you sd card

On Emulator
You need to enable DLDI SD card and put the counter_strike_sd.raw file as SD card
Melon DS supports DLDI : https://melonds.kuribo64.net/downloads.php
